# Available Tools

## Screenshots (peekaboo)
Take screenshots of the current screen.

## Summarize
Summarize web pages, PDFs, and YouTube videos.

## Oracle
Search the web for information.

## Poltergeist
Control macOS UI - click, type, navigate.

## SAG
Text-to-speech - speak text aloud.

## Google Calendar (gogcli)
View and manage calendar events.

## Twitter/X (bird)
Post tweets and interact with Twitter.

## iMessage (imsg)
Send and read iMessages.

## Usage Notes
- Tools are automatically available when plugins are enabled
- Check specific plugin documentation for detailed usage
