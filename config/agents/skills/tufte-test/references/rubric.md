# Tufte Test Rubric

This rubric is broken into seven rules.

## The seven rules

1. `rules/01-data-ink-ratio.md`
2. `rules/02-no-3d-effects.md`
3. `rules/03-direct-labeling.md`
4. `rules/04-axis-readability.md`
5. `rules/05-muted-purposeful-color.md`
6. `rules/06-integrated-annotations.md`
7. `rules/07-graphical-integrity.md`

## How to use

- Start with `SKILL.md` for the compact review workflow.
- Use these rule files only when you need more detail on a specific failure.
- Keep reviews strict but practical.

## Pass rule

A chart passes only if all seven rules pass.

## Minimal pass/fail summary template

```yaml
pass: true
summary: "Clean, readable, and honest. The chart uses restrained color, direct labeling, and a clear takeaway."
```
