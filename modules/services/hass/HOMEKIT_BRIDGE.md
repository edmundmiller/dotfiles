# HomeKit Bridge pairing / Siri scene wrappers

Home Assistant is the automation source of truth. Apple Home/HomePods should only call thin wrapper scenes that toggle Home Assistant script accessories.

## HA-side config

The HomeKit bridge is declared in `modules/services/hass/default.nix`:

- HomeKit bridge advertises the NUC LAN IP: `192.168.1.222`
- HomeKit bridge TCP port: `21063`
- NixOS firewall must allow TCP `21063`
- Exposed HomeKit accessories are HA scripts named:
  - `HA Good Night`
  - `HA In Bed`
  - `HA Sleep`
  - `HA Good Morning`
  - `HA Get Ready For Bed`
  - `HA Mid-morning`
  - `HA Sundown`
  - `HA Leave Home`
  - `HA Arrive Home`

Do not force a deterministic HomeKit PIN. Use the PIN/QR generated by Home Assistant.

## If pairing needs to be reset

1. In Home Assistant, go to **Settings → Devices & services → HomeKit Bridge**.
2. Remove the existing HomeKit Bridge entry if it is stale or paired to the wrong Apple Home.
3. Restart/redeploy Home Assistant:
   ```sh
   hey nuc
   ```
4. In Home Assistant, open **Settings → Devices & services → HomeKit Bridge** and pull the generated PIN/QR from the bridge card/configure screen or Home Assistant notification.
5. On the iPhone, temporarily disable Tailscale/VPN/exit-node routing and join normal home Wi-Fi.
6. In Apple Home, switch to **Moni’s Dream House**.
7. Add accessory → **More Options…** → select **Home Assistant Bridge**.
8. Enter the generated Home Assistant PIN or scan the QR code.

If Apple Home can discover the bridge but hangs on **Connecting…**, verify:

```sh
ssh nuc 'sudo ss -ltnp | grep 21063'
nc -vz 192.168.1.222 21063
```

The first command should show Home Assistant listening on port `21063`; the second should succeed from a LAN machine. If `nc` times out, check the NixOS firewall and local/VPN routing.

## Apple Home scene setup

After pairing the bridge into **Moni’s Dream House**, recreate Apple Home scenes as thin wrappers only. Do not put direct device actions in Apple Home scenes.

| Apple Home scene    | Only action inside scene    |
| ------------------- | --------------------------- |
| `Good Night`        | `HA Good Night = On`        |
| `Good Morning`      | `HA Good Morning = On`      |
| `In Bed`            | `HA In Bed = On`            |
| `Sleep`             | `HA Sleep = On`             |
| `Get Ready For Bed` | `HA Get Ready For Bed = On` |
| `Mid-morning`       | `HA Mid-morning = On`       |
| `Sundown`           | `HA Sundown = On`           |
| `Leave Home`        | `HA Leave Home = On`        |
| `Arrive Home`       | `HA Arrive Home = On`       |

HomePod flow should be:

```text
HomePod Siri → Apple Home scene → HA script accessory → Home Assistant routine
```
