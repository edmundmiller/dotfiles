# Retired Hermes Agent Module

`modules.agents.hermes` is intentionally a stub. Do not enable it.

Use:

- `modules.agents.hermes-desktop` for interactive desktop/user Hermes setup.
- `services.hermes` / upstream `services.hermes-agent` for NUC gateway profiles.

This stub exists only to fail clearly if old host config reintroduces the ambiguous `modules.agents.hermes.enable = true` path.
